# Как посмотреть сайт К Студии

## Самый простой способ — двойной клик (работает сразу)

1. Откройте папку `08_DEV` на рабочем столе
2. Дважды кликните на файл **`index.html`**
3. Сайт откроется в вашем браузере (Chrome, Edge, Safari и т.д.)

> ⚠️ При открытии через двойной клик шрифты Google Fonts загрузятся только при наличии интернета. Без интернета сайт откроется, но с системными шрифтами.

---

## Через VS Code — рекомендуется (лучший опыт)

Этот способ даёт правильные пути к файлам и живое обновление при редактировании.

### Шаг 1 — Установите расширение Live Server
1. Откройте VS Code
2. Нажмите `Ctrl+Shift+X` (Windows) или `Cmd+Shift+X` (Mac) — откроется панель расширений
3. Введите в поиске: **Live Server**
4. Нажмите **Install** (от автора Ritwick Dey)

### Шаг 2 — Откройте папку сайта
1. В VS Code: `File → Open Folder...`
2. Выберите папку `08_DEV` (она внутри `beauty-salon-website` на рабочем столе)

### Шаг 3 — Запустите Live Server
1. В нижней строке VS Code найдите кнопку **Go Live** (справа внизу)
2. Нажмите её — браузер автоматически откроется с адресом `http://127.0.0.1:5500`
3. Сайт работает корректно со всеми относительными путями!

## Деплой на хостинг (когда будете готовы)

Так как сайт — чистый HTML/CSS/JS без сборки, его можно загрузить **куда угодно**:

| Хостинг | Как загрузить | Цена |
|---------|--------------|------|
| **Vercel** | Перетащить папку на vercel.com/new | Бесплатно |
| **Netlify** | Перетащить папку на app.netlify.com | Бесплатно |
| **GitHub Pages** | Загрузить в репозиторий | Бесплатно |
| Любой хостинг с FTP | Загрузить файлы через FTP-клиент | По тарифу |

Рекомендуем **Vercel** или **Netlify** — загрузка за 2 минуты, HTTPS включён.
